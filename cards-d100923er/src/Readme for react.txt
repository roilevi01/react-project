# Watch Cards(Cards project)

Welcome to the "Watch Cards" project! This project is designed for creating and managing digital business cards for watch shops and professionals.

## Project Description

"Watch Cards" is a tool that allows users to create, manage, and share digital business cards. Each card includes information such as name, role, contact details, and social media links. The project offers a user-friendly interface that makes creating and editing cards easy.

## Key Features

- **Create Digital Business Cards**: Create custom cards with all the necessary information.
- **Manage Cards**: Easily edit, update, and delete cards.
- **Share Cards**: Share cards via email, social media, or QR code.
- **Custom Design**: Choose and customize different designs to suit your needs and preferences.
- **Registration and Login**: A system for registering on the site and a login page.
- **Admin Accounts**: Create admin accounts with advanced management permissions.
- **Business and Private Accounts**: Create business and private accounts with different options.

## Technologies

- **Node.js**: Server-side framework.
- **React**: JavaScript library for building the user interface.
- **JavaScript**: Main programming language.
- **MUI (Material-UI)**: Library for building the user interface, including pre-built components and icons.
- **CSS**: Styling and designing the cards and application.
- **Postman**: Tool for API testing and development.
- **React Router DOM**: Library for handling routing in React applications.

## Additional Tools and Libraries

- **MUI Icons**: Used for adding icons to the application.
- **React Hooks**: Used for managing state and lifecycle methods in functional components.

## Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/username/watch-cards.git
    cd watch-cards
    ```

2. **Install dependencies**:
    ```bash
    npm install
    ```

3. **Start the local server**:
    ```bash
    npm start
    ```

## Usage

After installation and starting the local server, you can access the application in your browser at `http://localhost:3000`.

### Usage Example

1. Access the application interface.
2. Click on the "Create New Card" button.
3. Fill in the required details (name, role, contact details, etc.).
4. Choose a custom design and click "Save".


### Registration and Login System Example

1. Navigate to the Sign Up page and register as a new user.
2. Log in using the Login page.
3. Go to the account management page to create new accounts and manage existing accounts.
